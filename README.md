Логический калькулятор 

  
  

Настольное приложение, разработанное на платформе WPF для анализа и вычисления логических функций булевой алгебры. Инструмент предназначен для студентов и специалистов, работающих в области дискретной математики и теории цифровых автоматов. 
Ключевые возможности 

     Построение таблиц истинности по номеру функции с автоматическим расчетом совершенных дизъюнктивной (СДНФ) и конъюнктивной (СКНФ) нормальных форм.
     Анализ логических формул с поддержкой полного набора базовых логических операций и скобок для задания приоритета.
     Сравнение функций на эквивалентность с возможностью ввода данных как в виде номера, так и в виде формулы. В случае неэквивалентности предоставляется контрпример.
     Интуитивный интерфейс, разделенный на три функциональные вкладки для удобства работы.

Руководство пользователя 

Приложение разделено на три функциональные вкладки, каждая из которых предназначена для определенной задачи. 
1. Вкладка "Таблица по номеру" 

Позволяет сгенерировать таблицу истинности для функции, заданной её номером в векторе Перрона. 

     "Количество переменных": Введите число от 1 до 20.
     "Номер функции": Введите десятичный номер функции.
     Результат: Отображается таблица истинности, а также совершенные ДНФ и КНФ с статистикой по количеству литералов и термов.
     

2. Вкладка "Таблица по формуле" 

Позволяет проанализировать функцию, заданную в виде логической формулы. 

     "Введите формулу": Используйте переменные вида x1, x2, ... и следующие операторы:
         ! — Отрицание (НЕ)
         & — Конъюнкция (И)
         | — Дизъюнкция (ИЛИ)
         ^ — Исключающее ИЛИ (XOR)
         -> — Импликация
         = — Эквивалентность
         
     Пример формулы: !(x1 & x2) | (x3 & !x4)
     Результат: Строится таблица истинности для заданной формулы, вычисляются ее СДНФ и СКНФ.
     

3. Вкладка "Сравнение функций" 

Позволяет определить, являются ли две функции эквивалентными. 

     "Функция 1" и "Функция 2": Введите номера функций или логические формулы в соответствующие поля.
     Результат:
         Если функции эквивалентны, появится соответствующее сообщение.
         В противном случае будет выведен контрпример — набор значений переменных, при котором результаты функций не совпадают.
